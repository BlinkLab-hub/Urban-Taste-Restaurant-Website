# Urban Taste Restaurant Website

A modern and responsive restaurant website built using HTML and CSS.

## Overview

Urban Taste is a restaurant website designed to provide visitors with an elegant and user-friendly dining experience. The project focuses on responsive design, clean layouts, and intuitive navigation across different screen sizes.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern navigation bar
- Animated hamburger menu
- Hero section with call-to-action buttons
- Clean and organized layout
- CSS animations and transitions

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- CSS Animations
